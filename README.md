## Getting Started

### Installing

`SSH`

```
git clone git@github.com:Raoun4136/chatbot_lab.git
```

or

`HTTPS`

```
git clone https://github.com/Raoun4136/chatbot_lab.git
```

then module install

```
npm install
```

### Starting

`In localhost`

```
npm run start
```

`To Deploy`

```
npm run deploy
```
